# Contributing to Clonnect

## Code Style

### Language Convention
- **Code**: All variable names, function names, class names, and module names in English
- **Comments**: English preferred for new code. Existing Spanish comments are acceptable — do not change them in bulk
- **User-facing strings**: Spanish (the product is primarily Spanish-speaking market)
- **Documentation**: English (for international auditors and collaborators)
- **Git commits**: English

### Formatting
- Python: Follow PEP 8
- TypeScript/React: Follow ESLint config
- Max line length: 120 characters
