PASOS DE SETUP COMPLETADOS
--------------------------

1. Clone Repository:
   git clone https://github.com/manelbertran-arch/clonnect-creators.git
   cd clonnect-creators

2. Configuracion .env:
   OPENAI_API_KEY=demo_key_for_testing
   ANTHROPIC_API_KEY=
   LLM_PROVIDER=openai
   DATA_PATH=./data
   API_HOST=0.0.0.0
   API_PORT=8000
   DASHBOARD_PORT=8501

3. Instalacion Dependencias:
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

TIPOS DE INTENT
---------------
GREETING - Saludo inicial - greet_and_discover
INTEREST_SOFT - Interes suave - nurture_lead
INTEREST_STRONG - Alta intencion compra - close_sale
OBJECTION - Objecion/duda - handle_objection
FEEDBACK_POSITIVE - Feedback positivo - thank_and_upsell
FEEDBACK_NEGATIVE - Feedback negativo - apologize_and_resolve
SUPPORT - Solicitud soporte - provide_support
QUESTION - Pregunta general - answer_question
SPAM - Spam/irrelevante - ignore_or_block
ESCALATION - Requiere humano - escalate_to_human
OTHER - No clasificable - generic_response

API ENDPOINTS
-------------
GET  /health                      Health check
GET  /docs                        Swagger UI
GET  /api/creators/{id}           Get creator config
GET  /api/creators/{id}/products  List products
POST /api/dm/process              Process incoming DM
POST /api/webhook/instagram       Instagram webhook

COMANDOS UTILES
---------------
# Iniciar API
cd /home/user/clonnect-creators
source venv/bin/activate
uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload

# Iniciar Dashboard
streamlit run dashboard/app.py --server.port 8501

# Ejecutar tests
pytest tests/ -v

# Ver logs
tail -f api.log
tail -f dashboard.log
